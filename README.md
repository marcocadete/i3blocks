## Battery  

### Dependencies
* acpi
* fontawesome

### Installation  
* Copy the battery script into your directory of choice, e.g. ~/.config/i3blocks/
* Give it execution permission (`chmod +x battery`)
* Add the following blocket to your i3blocks config file:  
```ini
[battery]  
command=$SCRIPT_DIR/battery  
interval=5  
markup=pango  
```
